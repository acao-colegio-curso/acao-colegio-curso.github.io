Desenvolvimento Web

Projeto Blog Institucional, visa introduzir conceitos de programação para os alunos, focando no aprendizado e entendimento de como funciona os sistemas que utilizamos assim como ter uma base de como esses são desenvolvidos, para que assim se torne usuários mais críticos e consigam resolver problemas ao utilizar tais meios.

Abaixo se encontra os conteúdos que iremos ver ao longo do nosso curso, que será dividido em seis módulos, sendo o módulo final o projeto prático da construção deste blog.
